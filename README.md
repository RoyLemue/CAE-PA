# CAE-PA
A project to read xml-based recipes from COMOS and control module based opcua-servers.

## Dependencies

Python 3.5
Django 1.10
PyQt5 (OpcUa-Client)
Opcua-client
freeopcua

## Contains

Bootstrap 3.3.7
Jquery 3.2.1

# Beschreibung

Die client.py in Main startet die beiden Server und die passenden Clients und gibt die States der Dienste aus.
